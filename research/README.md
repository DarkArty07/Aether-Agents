# Aether Agents — Research Vault

This directory is an [Obsidian](https://obsidian.md)-flavored knowledge base for persisted web research generated by Etalides, the Researcher Daimon.

## Structure

Each research file follows the naming convention:
```
YYYY-MM-DD-HHMM-topic-slug.md
```

Files contain YAML frontmatter with metadata and wikilinks `[[...]]` for cross-referencing related research.

## How It Works

1. **Etalides** performs deep web research and persists findings here
2. **YAML frontmatter** captures metadata: date, author, depth, confidence, model, links used/budget, tags
3. **Wikilinks** connect related research files for navigation in Obsidian or any wikilink-aware editor
4. **INDEX.md** provides a manually-maintained or auto-generated index of all research

## Opening in Obsidian

1. Open Obsidian → "Open folder as vault"
2. Select this `research/` directory
3. Obsidian will automatically recognize the wikilinks and tags

## Code Research

Code research (investigating project codebases) is delivered directly to Hermes as a response and is NOT persisted here. This vault is for web research only — the kind of knowledge worth indexing and reusing.

## Conventions

- File names: lowercase, hyphenated, URL-safe
- Tags in frontmatter: use bracketed lists like `[framework, security, api]`
- Confidence levels: `high`, `medium`, `low`
- Depth modes: `fast` (5 actions), `standard` (10 actions)
- Always save, even if results are incomplete or negative
